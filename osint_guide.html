Введение в OSINT

Что такое OSINT?

Open Source Intelligence (OSINT) — сбор, анализ и обработка общедоступной информации из открытых источников.

Источники OSINT:
  • Социальные сети и форумы
  • Поисковые системы
  • Публичные базы данных
  • Новостные сайты
  • Государственные реестры
  • Метаданные файлов
Применение OSINT:
  • Кибербезопасность
  • Журналистские расследования
  • Проверка информации
  • Конкурентная разведка
  • Поиск пропавших людей
  • Борьба с мошенничеством
Важно! Все действия должны выполняться в рамках закона и этических норм. Не нарушайте приватность и права людей.

Snoop Project - мощный инструмент поиска

Snoop Project - инструмент для автоматического поиска аккаунтов пользователя по никнейму на 800+ платформах.
Установка и использование Snoop Project:
Windows CMD 
# Скачайте архив с GitHub: https://github.com/snooppr/snoop
# Распакуйте в удобную папку
# Откройте командную строку в папке со Snoop
Команды Snoop 
# Базовый поиск по никнейму
snoop.exe username

# Поиск с фильтром по стране (Россия)
snoop.exe -f admin -t 9

# Поиск с сохранением в HTML
snoop.exe username --html

# Поиск только в топ-сайтах
snoop.exe username --top

EXIF анализ 
# Основные EXIF поля для анализа:
- Camera Make/Model (производитель/модель камеры)
- Date/Time Original (дата и время съемки)
- GPS Coordinates (координаты GPS)
- Software (программа обработки)

- ISO Speed (чувствительность ISO)
- Focal Length (фокусное расстояние)

Создание OSINT отчета

Структура OSINT отчета:

Титульный лист - название исследования, дата, автор
Исполнительное резюме - краткие выводы и ключевые находки
Цели и задачи - что искали и зачем
Методология - какие инструменты и методы использовались
Результаты исследования - детальные находки по каждому заданию
Анализ и выводы - интерпретация найденной информации
Рекомендации - предложения по дальнейшим действиям
Приложения - скриншоты, таблицы, схемы связей

Визуализация данных OSINT

Рекомендуемые инструменты для визуализации:
Maltego Профессиональная платформа для анализа связей
Gephi Открытая платформа для визуализации сетей
Lucidchart Онлайн-инструмент для создания диаграмм
Miro Интерактивная доска для мозгового штурма
Типы визуализации для OSINT:
Граф связей - показывает отношения между людьми, аккаунтами, организациями
Временная шкала - хронология событий и активности
Географическая карта - местоположения и перемещения
Матрица данных - сравнение характеристик объектов
Советы по оформлению отчета:
  • Используйте четкую структуру и заголовки
  • Добавляйте скриншоты с пояснениями
  • Указывайте источники и время сбора данных
  • Соблюдайте конфиденциальность персональных данных
  • Делайте выводы на основе фактов, а не предположений
Этические принципы OSINT:
  • Используйте только публично доступную информацию
  • Не нарушайте законы о защите персональных данных
  • Не используйте полученную информацию во вред
  • Проверяйте достоверность информации из нескольких источников
  • Документируйте процесс сбора данных
